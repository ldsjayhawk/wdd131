Site Plan

Site Name: Franchise GM Tools

The site name was chosen because this is going to be a site that contains tools to support a current year-round fantasy baseball site that I am currently in. The league is called FGM or Franchise GM. The intention is to further develop this site to refine these tools for actual use.

Site Purpose:

The purpose of the site is to provide tools that will augment those already provided by the site we play at.

• The sign-up provides the opportunity to sign up and provide needed info to the commissioner.

• The draft page provides the opportunity to automate draft functions with the scope of players narrowed to only those who can be selected.

• The roster page displays information about the players on a team's roster.

Scenarios

• Where can I sign-up?

• Where can I particpate in the draft?

• Where can I see my roster?

Color Schema

• Green (#5a8b39) will be used for the main header and buttons. It is a green color close to the color of the grass found at a baseball field.

• White will be used for the background

• Black will be used for smaller headers and paragraph fonts

Typography

•Main heading font: Arial

•Subsequent headings and paragraph font: Figtree
